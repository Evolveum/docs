Prerequisites

MidPoint is a Java application, therefore it requires Java platform to run on. Java 11 is a required platform for MidPoint {{ page.release-version }}{{ include.release-version }}. Java platform distributions based on OpenJDK are recommended. Other JDK builds (e.g. Oracle) may work as well, but those have limited support for midPoint (see release notes for the details).

JAVA_HOME environment variable should point to the Java installation path.
