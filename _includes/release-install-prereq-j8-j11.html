Prerequisites

MidPoint is a Java application, therefore it requires Java platform to run on. Java 11 is a recommended platform for MidPoint {{ page.release-version }}. Java platform distributions based on OpenJDK are recommended. Other JDK builds (e.g. Oracle) may work as well, but those have limited support for midPoint (see release notes for the details).

MidPoint {{ page.release-version }} may run on Java 8 platform. However Java 8 support is deprecated. If you choose to use Java 8, it is strongly recommended to migrate to Java 11 soon. As the situation with free updates to Oracle Java platform is not clear support for Java 8 may be removed even before the lifetime of this release is over.

JAVA_HOME environment variable should point to the Java installation path.
