= How to develop connectors using SCIMREST framework
:page-nav-title: Develop SCIMREST connectors
:page-display-order: 1200
:page-description: TBD
:page-keywords: TBD

This section provides two development guides for building connectors with the Polygon SCIMREST framework.

The xref:/connectors/scimrest-framework/develop/rest/[] section covers connector development for generic REST APIs.
It walks through initial setup, schema definition, search implementation, and—for read-write connectors—create, update, and delete operations using Groovy DSLs.

The xref:/connectors/scimrest-framework/develop/scim/[] section covers connector development specifically for SCIM 2.0 compliant services.
It follows the same progression but takes advantage of the built-in SCIM support of the framework, including the automatic schema discovery, and native handling of SCIM search filters and PATCH/PUT operations.

Both guides address read-only and read-write connector development.
If you are unsure which guide to follow, use the SCIM 2.0 guide when the target service exposes a SCIM 2.0 compliant API;
otherwise, use the SCIMREST connector guide.
